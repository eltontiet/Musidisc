# Musidisc

A simple discord music bot.

## Commands

The bot can be interacted with using the "slash" commands in discord. This list can be seen by inputting '/' into discord.

- `/play [title]` plays the first result when searching youtube
- `/search [title]` returns a list of videos from youtube that can then be selected to play
- `/stop` stops playback
- `/pause` pauses playback
- `/resume` resumes playback
- `/skip` skips current song
- `/np` displays the currently playing song
- `/queue` returns the queue
